Challenge Flow: Where does "Challenge a Friend" live?

Before we design the screens, where should the entry point be?

A

On the Train Tab (Game List)

Each game card gets a small "challenge" icon/button. Tap it to start a challenge for that specific game. The game list becomes both "play solo" and "challenge a friend" in one place.

B

Dedicated "Challenge" Button on Home

A prominent "Challenge a Friend" card on the Home tab. Tapping it opens a game picker, then flows into the challenge. Keeps the Train tab clean for solo play.

C

Inside the Compete Tab

Add a "Challenge a Friend" section alongside leaderboards. Makes sense thematically — competing is what this tab is for. Game picker built into the challenge flow.

D

Multiple Entry Points

Challenge icon on each game card in Train tab AND a "Challenge" section in Compete tab. More discoverable, but more UI surface area to maintain.
